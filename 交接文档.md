## 二、项目架构

```
浏览器 → Nginx(80) → Node.js(3000) → PostgreSQL(5432)
              │
   docker compose 三件套（restart: unless-stopped）
```

### 文件结构
```
~/TsToolsPlatform/
├── docker-compose.yml    # 三服务编排（app + postgres + nginx）
├── Dockerfile            # 多阶段构建（node:22-alpine）
├── .env                  # 环境变量（飞书OAuth + 数据库密码）
├── nginx/nginx.conf      # Nginx 反向代理配置
├── server/db/migrations/ # 数据库迁移脚本
└── server/                # 后端代码
```

## 三、日常运维命令

### 启动 / 停止
```bash
cd ~/TsToolsPlatform

# 首次构建（已执行过一次，以后不需要加 --build）
docker compose up -d --build

# 日常重启（不加 --build，几秒完成）
docker compose up -d

# 停止
docker compose stop

# 查看状态
docker compose ps

# 查看日志
docker compose logs -f app
docker compose logs -f postgres
```

### 更新代码
```bash
cd ~/TsToolsPlatform
git pull                        # 拉取最新代码
docker compose up -d --build    # 重新构建并启动
```

### 数据库管理
```bash
# 进入 PostgreSQL
docker compose exec postgres psql -U postgres -d tscameratools

# 查看用户
SELECT * FROM users;

# 查看登录会话
SELECT * FROM sessions;

# 查看设备历史
SELECT * FROM device_history;
```

### 备份
```bash
# 备份数据库
docker compose exec postgres pg_dump -U postgres tscameratools > backup_$(date +%Y%m%d).sql

# 恢复数据库
docker compose exec -T postgres psql -U postgres tscameratools < backup.sql
```
