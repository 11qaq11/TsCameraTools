# 终端 UI 修复计划

## 概述

修复 Reconnect 按钮被裁剪问题 + 添加 ADB Shell 中文输入警告。

---

## 问题分析

### 问题 1：Reconnect 按钮在 UI 外

`XtermTerminal.tsx` 的外层容器有 `overflow-hidden`，断连按钮放在 `termRef` div 下方。当终端内容填满时，按钮被推出可视区域。

### 问题 2：缺少中文输入警告

ADB Shell 不支持中文输入，输入中文会导致断连，但用户无提示。

---

## 修复方案

**文件：`src/components/terminal/XtermTerminal.tsx`**

### 修改 1：按钮改为覆盖层

将 Reconnect/Close 按钮从终端下方移到终端区域内，使用 `absolute` 定位覆盖在终端上方：

```tsx
<div ref={termRef} className="flex-1 relative bg-[var(--color-background)]" />
{(status === 'disconnected' || status === 'error') && (
  <div className="absolute inset-0 flex items-center justify-center bg-[var(--color-background)]/80 backdrop-blur-sm">
    <div className="flex flex-col items-center gap-3 rounded-xl border border-[var(--color-border)] bg-[var(--color-card-bg)] p-6 shadow-lg">
      <p className="text-sm text-[var(--color-text-secondary)]">连接已断开</p>
      <div className="flex gap-2">
        <button onClick={handleReconnect} className="...">
          <RefreshCw size={14} /> Reconnect
        </button>
        <button onClick={handleClose} className="...">
          Close
        </button>
      </div>
    </div>
  </div>
)}
```

### 修改 2：ADB Shell 中文输入警告

在标题栏下方、终端上方添加条件提示条（仅 ADB 类型显示）：

```tsx
{type === 'adb' && status === 'ready' && (
  <div className="px-4 py-1.5 text-xs text-[var(--color-accent-orange)] bg-[var(--color-accent-orange)]/10 border-b border-[var(--color-border)]">
    Shell 暂不支持中文输入，请使用英文
  </div>
)}
```

---

## 实施顺序

1. 修改 Reconnect 按钮为 absolute 覆盖层
2. 添加 ADB 中文输入警告提示
3. `npm run web:build` 验证

---

## 验证

- 断连后，Reconnect 按钮在终端区域内居中显示（半透明遮罩背景）
- ADB Shell 连接成功后，标题栏下方显示橙色警告条
- 本地终端不显示警告条
