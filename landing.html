// 核心功能

为影像开发团队量身打造的专业工具集

ADB 设备管理

自动检测 ADB 安装、设备连接、Root/Remount 操作。一键安装 ADB 工具。

交互式 Shell 终端

基于 xterm.js 的 Hyper 风格终端，支持命令历史、Ctrl+R 搜索、中文输入。

飞书 OAuth 登录

集成飞书企业账号 OAuth 2.0 认证，支持中科创达团队统一登录。

命令历史记录

最多保存 300 条命令历史，支持上下键导航和 Ctrl+R 反向搜索。

双模式运行

支持 Web 模式（推荐）和 Electron 桌面模式，满足不同场景需求。

TypeScript 全栈

React 19 + TypeScript + Vite 8 + Tailwind CSS 4，现代化技术栈。

// 快速开始

三步启动开发环境

1

克隆项目

$ git clone https://github.com/11qaq11/TsCameraTools.git
$ cd TsCameraTools
2

安装依赖

$ npm install
3

启动开发服务器

$ npm run web:dev

✓ VITE ready in 325ms
✓ 服务器运行在 http://localhost:3000
