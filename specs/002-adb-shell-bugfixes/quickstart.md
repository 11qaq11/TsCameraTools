# Quickstart: ADB Shell Bug Fixes

**Created**: 2026-07-04

## 验证场景

### V1: 输出格式化 (BUG-001)

**前置条件**: ADB 已安装，设备已连接
**步骤**:
1. 启动程序，连接设备
2. 在终端输入 `ls` 并回车
3. 观察输出格式

**预期**: 每行输出前有一个空格，与 prompt 对齐

---

### V2: Ctrl+C 清空输入 (BUG-002)

**前置条件**: ADB 已安装，设备已连接
**步骤**:
1. 连接设备，打开终端
2. 快速输入 `abcde`
3. 按 Ctrl+C
4. 输入 `ls` 并回车

**预期**: 
- 显示 `^C` 并换行
- `ls` 命令正常执行，不会出现 `abcde` 的残留

---

### V3: 终端复制 (BUG-003)

**前置条件**: ADB 已安装，设备已连接
**步骤**:
1. 连接设备，执行 `echo hello world`
2. 用鼠标选中 `hello`
3. 在其他应用粘贴

**预期**: 粘贴内容为 `hello`

---

### V4: 最后一行可见 (BUG-004)

**前置条件**: ADB 已安装，设备已连接
**步骤**:
1. 连接设备，打开终端（非全屏模式）
2. 执行 `ls -la /system` 产生大量输出
3. 观察最后一行

**预期**: 最后一行（prompt 行）始终可见，不被截断

---

### V5: 回归测试

**步骤**:
1. 执行所有 V1-V4 场景
2. 验证命令历史（上下方向键）仍然正常
3. 验证 Ctrl+R 搜索仍然正常
4. 验证中文输入仍然正常

**预期**: 所有现有功能不受影响
