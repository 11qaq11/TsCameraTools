# UX 完成度 Checklist

**Feature**: specs/006-ui-redesign-terminal
**Created**: 2026-07-07
**用途**: 前端实现完成后逐项验证 UI/UX 质量

## 视觉质量

- [ ] 背景色为 #F8FAFC（非纯白 #FFF，减少视觉疲劳）
- [ ] 文字主色为 #0F172A（深色，确保可读性）
- [ ] 主色调为 #2563EB（蓝色，用于按钮、链接、高亮）
- [ ] 卡片背景为 #FFFFFF，边框为 #E2E8F0
- [ ] 所有图标来自 lucide-react，风格统一
- [ ] 无 emoji 作为结构性图标
- [ ] 字体为 Plus Jakarta Sans（主字体）+ JetBrains Mono（等宽）
- [ ] 色彩对比度全部 >= 4.5:1（WCAG AA）

## 导航栏

- [ ] 默认展开状态，宽度 240px
- [ ] 显示 Logo + 应用名称 + 菜单项 + 版本号
- [ ] 折叠按钮可见，点击后收缩到 64px
- [ ] 折叠状态下仅显示图标，鼠标悬停显示 tooltip
- [ ] 当前路由对应菜单项高亮（蓝色背景）
- [ ] 展开/折叠动画流畅（200-300ms）
- [ ] 菜单项点击可正确切换页面

## 终端体验

- [ ] 点击 Connect 后终端区域加载，显示 loading 状态
- [ ] ttyd 终端正常渲染（xterm.js 画布）
- [ ] 终端背景为白色，文字为深色（Light 主题）
- [ ] 中文输入法（IME）可正常使用
- [ ] Ctrl+C 可中断当前命令
- [ ] 终端窗口 resize 时自动调整列数/行数
- [ ] ttyd 进程退出时显示断开提示，非白屏

## 设备卡片

- [ ] 设备型号和序列号清晰显示
- [ ] 在线设备显示绿色状态标签
- [ ] 离线设备显示红色状态标签
- [ ] Root / Remount / Connect 按钮布局合理
- [ ] 离线设备的 Connect 按钮禁用（opacity-50）
- [ ] 操作按钮有 hover 状态反馈

## 响应式

- [ ] 1440px 布局正常（主断点）
- [ ] 1280px 布局正常
- [ ] 1024px 布局正常（最小支持宽度）
- [ ] 无水平滚动条（内容自适应）

## 交互反馈

- [ ] 所有可点击元素有 cursor: pointer
- [ ] 按钮 hover 有颜色变化反馈
- [ ] 加载中状态有 spinner 或 skeleton
- [ ] 错误状态有明确提示文字（非 console only）
